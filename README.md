### Bitespeed Identity Reconciliation Service

This is a Node.js + TypeScript + Express backend that implements the Bitespeed identity reconciliation logic using a PostgreSQL database.

- **Base URL**: [`https://codebyte-hspz.onrender.com`](https://codebyte-hspz.onrender.com)
- **Endpoint**: `POST /identify`

Send a JSON body with optional `email` and/or `phoneNumber` fields to this endpoint to test the service.
